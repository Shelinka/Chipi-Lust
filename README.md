# Chipi Lust

Chipi Lust is a World of Warcraft addon that reacts when you get bloodlust-style fatigue debuffs.

When triggered, it can:
- play a selected sound
- show a selected image/animation
- show a 40-second countdown timer

You can configure everything in-game per profile, including enable/disable, sound, image, size, position, and timer display.

## Install

Copy the addon folder to:

`World of Warcraft/_retail_/Interface/AddOns/Chipi Lust`

Reload/restart the game.

## Configure

Open:

Game Menu → Options → AddOns → Chipi Lust

Use **Test Effect** to preview your selected settings.
